1. 关于字符串的相关知识点

预备知识：

1.Java中用双引号引起来的都是字符串，不能作为其它数据类型来使用，如“1+2”输出结果不可能是3 （用单引号引起来的是字符）

2.Java中字符串作为string类的实例来处理，可以通过Java.lang包中的String类来创建字符串对象，以对象的方式处理会使字符串更灵活，方便

3.在声明字符串之后最好将其赋值，不然使用时会报错，声明方式： String s (s表示字符串的名称)



创建字符串：

创建字符串的两种方式：
1.将字符串作为对象来处理，像创建对象一样创建字符串（即通过类的方法，来实例化一个字符串对象）
2.创建一个字符串类型的常量，然后赋值

方式一的方法：

1.用字符数组创建字符串对象

1     char a[] = {'g','o','o','d'};
2     String s = new String(a);
3      //该方式等价于 String s = new String("good");

2.提取字符数组中的一部分来创建一个字符串对象

1     char a[] = {'s','t','u','d','e','n','t'};
2     String s = new String(a,2,4);
3      //该方式等价于 String s = new String("uden");
3      // 2表示从第二个字符开始截取，4表示往后截4个字符

3.直接创建String对象

1     String c = new String("abcd");
2     String d = new String("abcd");
3     System.out.println(c == d);  //False	
4     System.out.println(c.equals(d)); //True

c == d是false，因为c和d指向的不是一个引用，构造方法创建的字符串对象位于堆内存中，每次new都是创建不同的对象，因此c和d的引用不一致

方式二的方法：(仅一种)

1     String str1,str2;
2     str1 = "we are students";
3     str2 = "we are students";
4     System.out.println(a == b); //True	
5     System.out.println(a.equals(b));    //True

==是进行值的比较，对于原始类型来说，就是比较它们的值，对于引用类型来说，就是检查它们引用的是不是同一个对象

当同样的字符串再次被创建的时候，便不再创建新的字符串，这叫做“String interning”， 编译期的字符串常量是自动interned的（就是会维护一个常量池，如果池中有这个字符串，那么直接引用之，否则再创建）



连接多个字符串：

用“+”运算符实现， 用“+”连接其它数据类型时，会自动调用to string()方法，将其转化成字符串（前提：其中有一个或多个字符串，不考虑运算符优先级



获取字符串信息：

获取字符串长度：使用String类的length()方法

1     String str1 = "we are students";
2     int zise = str.length();

字符串查找：使用indexOf()或者lastIndexOf()方法
